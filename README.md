# Campus Placements

A collection of notes, questions, and study material to help prepare for campus placement interviews and exams.

## Repository Structure

- `Companies Questions/` : Practice problems and interview questions from specific companies.
- `Computer Fundamentals/` : Core CS fundamentals and quick revision notes.
- `DBMS/` : Database concepts, sample questions, and notes.
- `DSA/` : Data Structures and Algorithms resources, problem sets, and solutions.
- `Java Notes & Questions/` : Java language notes, common questions, and examples.
- `SOft Skills/` : Communication, resume tips, and interview etiquette.

## How to use

- Browse the folders above and open files relevant to the topic you want to study.
- Use the `DSA/` and `Companies Questions/` folders for hands-on practice and mock interviews.
- Add your own notes or solutions by creating new files or folders following the existing structure.
